# Simple Task List

This project is a simple Task Management System developed in Python and executed in Google Colab.

## Features
- Add tasks
- Remove tasks
- View tasks
- Machine Learning based priority recommendation

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Google Colab

## Data Storage
Tasks are stored in a CSV file (`tasks.csv`).

## How to Run

Open the notebook in Google Colab and run all cells.
